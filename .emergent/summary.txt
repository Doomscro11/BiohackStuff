<product_requirements>
The product, Peptimancer, is an AI-powered platform for designing novel peptide analogues. Initially, it was a simple generator based on a base molecule and modifications.

The platform's capabilities have been progressively expanded to meet enterprise needs:
- **Core Functionality**: Generate peptide analogues with specified modifications and exclusions.
- **Enterprise Features**:
    - **Secure Admin Panel**: A protected area for administrators to manage the application, featuring email OTP and TOTP-based 2FA, Role-Based Access Control (RBAC), and the ability to change runtime settings dynamically.
    - **Billing & Credits**: A complete system (using a mock provider, with Stripe integration planned) for managing user subscriptions, credit plans, credit consumption for generating analogues, and viewing transaction history.
    - **Admin Analytics**: A dashboard providing administrators with insights into usage patterns, including active users, analogues generated, credit flows, and popular modifications, powered by a nightly data collection job.
    - **PatentPulse Module**: A new vertical for mining and analyzing expired or near-expiry biotech patents to identify commercialization opportunities. This includes a data collector, a scoring engine, and a dedicated admin dashboard.
- **UI/UX Enhancements**: The main generation form was redesigned from a simple list into a compact, user-friendly accordion interface that groups options by pharmacokinetic intent and displays tier-gated features.
</product_requirements>

<key_technical_concepts>
- **Full-Stack Frameworks**: React for the frontend, FastAPI (Python) for the backend.
- **Database**: MongoDB for application data, including dynamic settings, user info, billing ledgers, and analytics.
- **Authentication & Security**: JWT for sessions, email OTP for login, and TOTP for Admin 2FA. Middleware enforces RBAC.
- **Billing**: A modular system with adapters for mock or real (Stripe) payment providers, handling subscriptions, credits, and webhooks.
- **Asynchronous Jobs**: Python scripts for nightly data rollups (analytics) and data ingestion (PatentPulse), with plans for cron scheduling.
- **Testing & Deployment**: Automated testing with Playwright (frontend) and a custom backend test agent. Failsafe deployment strategies including Git tagging, feature branching, and plans for canary releases.
- **Observability**: Error logging, system health checks, and plans for a Grafana dashboard.
</key_technical_concepts>

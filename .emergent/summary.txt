<product_requirements>
The product, Peptimancer, is an AI platform for designing peptide analogues. The primary goal of the work captured in this trajectory was to build and stabilize the **PatentPulse Partner Portal (Phase IXf+)**.

This feature enables the secure sharing of Reclaim Packs (curated patent opportunity reports) with external partners. The core requirements implemented were:
-   **Secure Link Generation:** Admins can create unique, expiring, and usage-limited share links for PDF/JSON reports. These links are signed using an HMAC secret.
-   **Policy Enforcement:** The system enforces policies such as link expiry (TTL), maximum download counts, and IP address allow-listing.
-   **Watermarking:** Shared PDFs are dynamically watermarked with the recipient's information. JSON files receive special metadata headers.
-   **Partner-Facing UI:** A simple, no-login page where partners can view report metadata and download the files, with clear error states for expired or revoked links.
-   **Admin Management UI:** A new panel in the admin dashboard to create, revoke, and monitor share links.
-   **Analytics:** The system tracks and logs events like link opens, downloads, and blocked access attempts.

Following the feature implementation, a new requirement was introduced: a full structural refactoring of the monorepo into a clean, domain-oriented layout without altering any external behavior.
</product_requirements>
